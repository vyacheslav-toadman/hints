Selectors

[test~="foo"] : Checks if the value of the attribute test contains the word foo no matter of the position. [test|="foo"] : Checks if the value of the attribute test starts with the word foo. [test^="foo"] : Checks if the value of the attribute test has the first 3 letters foo. [test$="foo"] : Checks if the value of the attribute test ends with the word foo.
